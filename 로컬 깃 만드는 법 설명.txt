인터넷에 git 설치 검색
https://git-scm.com/install/windows 
여기 사이트 들어가서 git 설치 (전부 next 누르고 설치 완료하면 됨)

윈도우 검색창에 git bash 입력
$git 입력했을 때 git이 정상적으로 설치되면 설명문이 뜰거임
아니면 다시 재설치 하면 됨

폴더 하나 만들어서 README.txt 파일 하나 만듬
오른쪽 마우스 클릭 후 git bash here 클릭하면 git bash 화면이 뜰거임
$git init 입력해서 .git 이라는 폴더 생성함 (숨김파일로 생성됨)
$git init (.git 폴더 생성 명령어임, 여기에 git으로 생성한 버전들의 정보와 원격저장소 주소 등이 적혀져있음)
.git 폴더를 깃 로컬저장소라고 부름

로컬에서만 버전관리를 할거면 github에 등록하지 않은 email, username을 입력해도 되지만 
github를 사용할 예정이라면 github의 계정정보를 동일하게 입력해야 한다.
$ git config --global user.email "jasen70853@naver.com"
$ git config --global user.name "hirubykim"

커밋에 추가할 파일을 선택
$git add README.txt

커밋에는 상세 설명을 적을 수 있음 
(내가 이파일을 왜 만들었는지 왜 수정했는지 알 수 있고 해당버전을 찾아 그 버전으로 바꾸기 쉽다)
$git commit -m "사이트 설명 추가"

>> 이렇게 하면 첫번째 커밋을 만든 것임

>> 파일 수정 후 두번째 커밋도 동일한 방식으로 하면 됨

>> 파일 수정 후 두번째 커밋을 만들었다고 가정하에 아래 진행함

커밋했던 깃의 로그 내역보기
$git log
(제일 최신순으로 로그 내역이 보이고 젤 아래에 맨 처음에 만든 깃 내역이 보임
그리고 로그 내역을 보면 commit에 커밋아이디가 있음)


첫번째 커밋으로 되돌리기 
$git checkout  afe21c25 (커밋 아이디 전부를 입력해도 되고 앞자리 7자리를 입력해도 됨)
그럼 head is now at  afe21c25 라는 텍스트가 보이면 성공한 것임

첫번째 커밋으로 되돌린 걸 다시 두번째로 되돌리기 (원복하기)
동일하게 $git checkout f95a328 를 사용해서 두번째로 원복할 수 있지만 
두번째 파일이 최신 파일이라는 점을 이용해서 
$git ckeckout - 도 사용할 수 있다 - 의 의미는 최신 커밋을 말한다.
